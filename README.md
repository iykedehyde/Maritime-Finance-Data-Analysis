# Maritime Finance & Data Analysis

Finance, controlling, and data analytics examples in a maritime asset management context using SQL, Python, Excel, and BI tools.

## Background
I have several years of experience in finance and controlling within
international maritime organisations. This repository demonstrates how I
approach business data, cost structures, and performance analysis using
SQL, Python, Excel, and BI tools.

## Contents
- SQL examples for cost and performance analysis
- Python notebooks for basic data preparation and analysis
- BI examples for management reporting

## Tools & Technologies
- SQL (Microsoft SQL Server / Azure SQL style)
- Python (pandas)
- Power BI
- Microsoft Excel

## Focus Areas
- Maritime cost structures
- Budget vs actual analysis
- Operational and financial performance transparency

This repository is intended as a practical demonstration of my analytical
approach and learning path in data analytics.
